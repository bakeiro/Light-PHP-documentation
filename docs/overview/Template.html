            # 🎨 Templates

            Light-PHP uses a very tiny and simple template engine, this is how to use it:

            Simply put the {{}} surrounding the variable name to output the value

            # Before:

            	<p><?php echo $user_name ?></p>
	<p><?php echo $user_address ?></p>

            # After:

            	<p>{{user_name}}</p>
	<p>{{user_addresss}}</p>
